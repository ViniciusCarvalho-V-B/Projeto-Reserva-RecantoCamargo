# Projeto-Reserva-RecantoCamargo
Um sistema em java para reserva de imoveis em aparecida com sistema de avaliacao proprietario hospede e cadastro de imoveis

O sistema contem um sistema de autenticacao basico para acesso ao painel de proprietario
O sistema contem um sistema de verificacao de data para alta temporada ou feriados onde o pre√ßo se altera (DataDinamica)
O proprietario pode cadastrar um novo imovel, remover um imovel,aprovar uma reserva e ver todos os imoveis cadastrados
O hospede precisa de identificar na hora da reserva apenas
O hospede pode ver todos os imoveis cadastrados no sistema
O hospede pode reservar qualquer imovel cadastrado
O hospede apos uma reserva pode avaliar como foi sua estadia
